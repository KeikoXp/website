Comandos

Eu possuo diversos comandos, então eu separei eles em categorias para você poder procurar melhor!

Social

Socialize com o pessoal do servidor!

  • ;ship <membro um> <membro dois>

    Veja a porcentagem do amor entre duas pessoas!

  • ;kiss <membro>

    De um beijinho em alguém!

  • ;slap <membro>

    De um tapa em alguém!

  • ;pat <membro>

    Faça carinho em alguém!

  • ;hug <membro>

    De um abraço beem forte em alguém!

Anything

Qualquer coisa que não seja relacionada a entreterimento.

  • ;suporte

    Entre no meu servidor e tire suas dúvidas, venha, você vai se divertir!

  • ;majorie

    Saiba mais sobre mim! tipo alguns detalhes técnicos.

  • ;invite

    Veja o meu link de convite para você poder me adicionar nos seus servidores!

  • ;votar

    Vote em mim e receba recompensas no sistema de duelo!

  • ;connection <ID do usuário>

    Cria uma conexão com determinado usuário.

    Apenas desenvolvedores podem executar este comando.

Moderation

Modere o seu servidor de forma fácil!

  • ;ban <ID do usuário>

    Bana um usuário do servidor ou fora!

    O BOT e o usuário que executou o comando devem ter a permissão ban_members para executar este comando.

  • ;kick

    Expulse quem não cumprir as regras!

    O BOT e o usuário que executou o comando devem ter a permissão kick_members para executar este comando.

  • ;mutar

    Mute as pessoas que falam demais...

    O BOT e o usuário que executou o comando devem ter a permissão manage_roles para executar este comando.

    É preciso definir um cargo para executar este comando. Veja o comando ;setar Muterole para mais informações.

  • ;desmutar

    Desmute as pessoas que falaram demais...

    O BOT e o usuário que executou o comando devem ter a permissão manage_roles para executar este comando.

    É preciso definir um cargo para executar este comando. Veja o comando ;setar Muterole para mais informações.
